# Calendario FUNDAE 2026 — Fechas Clave y Plazos

> Versión: v2026.1 · Actualizado: Marzo 2026

Este calendario recoge todas las fechas relevantes para la gestión de formación bonificada FUNDAE durante el ejercicio 2026.

---

## Plazos permanentes (aplican todo el año)

| Plazo | Detalle | Empresas afectadas |
|-------|---------|-------------------|
| **7 días naturales antes del inicio** | Comunicación de inicio de acción formativa | Empresas de 10+ trabajadores |
| **2 días naturales antes del inicio** | Comunicación de inicio de acción formativa | Empresas de 1-9 trabajadores |
| **3 meses tras finalizar** | Comunicación de finalización | Todas las empresas |
| **15 días desde comunicación a RLT** | Plazo para que la RLT emita informe | Empresas con RLT |
| **4 años** | Plazo de conservación de documentación (prescripción) | Todas las empresas |

---

## Fechas clave del ejercicio 2026

### Enero
- **1 de enero**: Se asigna el crédito de formación para el ejercicio 2026, calculado sobre la cotización por formación profesional de 2025.
- **Enero-Febrero**: Período óptimo para planificar el plan de formación anual y comunicar las primeras acciones formativas.

### Primer trimestre (enero-marzo)
- Mejor momento para iniciar la formación del año, especialmente para cursos largos que necesitan varios meses.
- Si la empresa no usó el crédito de 2025, ya lo ha perdido. Evitar repetir el error.

### Segundo trimestre (abril-junio)
- Período intermedio. Buen momento para evaluar el consumo de crédito y ajustar el plan formativo.
- Las empresas que no han iniciado formación aún tienen tiempo, pero conviene no demorarse.

### Tercer trimestre (julio-septiembre)
- **Atención**: Si no has iniciado formación este año, es el último trimestre cómodo para hacerlo.
- Julio-agosto: Menor actividad formativa por vacaciones, pero las comunicaciones de inicio se pueden hacer igualmente.
- **Septiembre**: Inicio del curso escolar. Pico de demanda de formación bonificada.

### Cuarto trimestre (octubre-diciembre)
- **Octubre-Noviembre**: Último momento recomendable para iniciar nuevas acciones formativas. Los cursos deben finalizar antes del 31 de diciembre para bonificar en este ejercicio.
- **Noviembre**: Si quedan cursos pendientes de finalizar, hacer seguimiento para asegurar que los participantes completan a tiempo.
- **31 de diciembre**: **FECHA LÍMITE ABSOLUTA**. Todo el crédito no utilizado se pierde. No es posible acumularlo para 2027.

---

## Recordatorios trimestrales recomendados

| Mes | Acción recomendada |
|-----|-------------------|
| Enero | Consultar crédito asignado. Planificar formación anual. |
| Marzo | Revisar primeras acciones formativas. Verificar comunicaciones. |
| Junio | Balance semestral: % de crédito consumido. Ajustar plan si necesario. |
| Septiembre | Últimas acciones formativas del año. Comunicar antes de octubre. |
| Noviembre | Seguimiento de cursos en marcha. Verificar que finalizarán a tiempo. |
| Diciembre | Comunicar finalizaciones pendientes. Aplicar bonificaciones en TC1. |
