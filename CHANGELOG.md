# Registro de Cambios (Changelog)

Todas las actualizaciones significativas de esta guía se documentan aquí. El formato sigue [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/) y el versionado [Semántico](https://semver.org/lang/es/).

## [v2026.1] — 2026-03-03

### Publicación inicial
- README principal con guía resumida del proceso de formación bonificada
- Guía completa del proceso paso a paso con errores comunes y soluciones
- Glosario de términos FUNDAE
- FAQ exhaustiva (30+ preguntas con respuestas detalladas)
- Calculadora de crédito FUNDAE en JavaScript
- Plantillas: comunicación a RLT, comunicación a trabajadores, checklist de auditoría, acuerdo con entidad formadora, encuesta de satisfacción
- Calendario FUNDAE 2026 con fechas clave y recordatorios
- Estadísticas sobre uso de formación bonificada en España

---

## Convención de versiones

- **vAÑO.X**: Versión principal del año, actualizada con cada cambio normativo
- Ejemplo: v2026.1 = primera versión del ejercicio 2026, v2026.2 = actualización tras cambio normativo

## Cómo se actualiza

Este repositorio se actualiza cuando:
1. FUNDAE publica cambios normativos
2. El Ministerio de Trabajo emite nuevas instrucciones
3. Se detectan errores en la información existente
4. Se añade nuevo contenido (guías, plantillas, herramientas)

Suscríbete a los [Releases](../../releases) de este repositorio para recibir notificaciones de cada actualización.
