# Guía completa de formación bonificada FUNDAE
{: .fs-9 }

Documentación, herramientas y datos abiertos sobre formación bonificada para empresas en España. Todo lo que un técnico de RRHH necesita para gestionar la bonificación FUNDAE.
{: .fs-6 .fw-300 }

<span style="font-size:12px;color:#71717a">Mantenido por <a href="https://www.ciberaula.com">CiberAula</a> · Desde 1997</span>

---

<div class="bib-tldr">
  <span class="bib-tldr__label">¿Qué es este repositorio?</span>
  <p>Un repositorio público con la documentación técnica que necesita cualquier empresa española para gestionar su <strong>crédito de formación bonificada ante FUNDAE</strong>: el proceso paso a paso, las preguntas frecuentes, una calculadora de crédito, el calendario de plazos, datos estadísticos del sector y plantillas de documentación.</p>
  <p>La versión canónica y completa de estos contenidos vive en <a href="https://www.ciberaula.com/biblia-formacion-bonificada/"><strong>La Biblia de la Formación Bonificada</strong></a> en ciberaula.com.</p>
</div>
